  16. WPHP
  17.  
  18. WhitePHP, or WPHP for short, a PHP framework like whitepaper that you can deal with it as you like. I was bored with big framework. It just gives me too much. Guess what, I dont't found the right framework even. So why not write a framework myself. This is how this framework was born. This framework learn from codeigniter, lazyphp. Thanks you all!
  19. I will keep this framework simple, secure and high efficient.
  20.  
  21. WhitePHP，即WPHP，是一个非常容易上手的PHP框架。因为现有的框架结构有点复杂，有时候是杀鸡用牛刀。所以在一天晚上，自己写了这个框架。此框架借鉴了CodeIgniter和Lazyphp，谢谢你们，谢谢开源。
  22. 我将会保持这个框架简单、安全、高效。
  23.  
  24. WPHP框架
  25. WPHP 是 WhitePHP 的缩写，它是一个简单高效的 PHP 框架。WhitePHP 极其简单，就像一张任你书写的白纸一样。
  26.  
  27. 该框架遵循 MVC 模式，MVC 各个部分可以从清晰的目录结构中辨别出来。即使没用过其它框架也会很快上手，因为此框架设计之初就是因为其他框架给的太多，定义了太多的函数，所以该框架尽可能的降低了学习成本。
  28.

  51. 注意：
  52. 0，方法不能和类名同名，文件不能和文件夹同名
  53. 1，文件名全部小写，类名首字母大写
  54. 2，系统函数均为下划线分割，类首字母大写，方法和函数命名保持一致
  55. 2,set_conf() 等函数的保留索引为 controller, action 
  56. 3，单入口文件，所有相对路径都是相对于前端控制器的
  57. 4，低耦合，高聚合
  58.  
  59. 系统级别常量表

  61. SYS_PATH                前段控制器所在目录
  62. INDEX_PAGE                入口文件即前端控制器文件名
  63. QUERY_STRING            网址中的查询字符串
  64. CUR_CONTROLLER            当前控制器名称
  65. CUR_ACTION                当前方法名称
  66.  
  67. 系统固定常量
  68. VERSION                    框架版本号
  69. LOG_PATH                 日志路径
  70. WPHP_GLOBAL_CONFIG_NAME 全局变量存储索引
  71.  
  72. 数据库配置常量
  73. IS_DB_ACTIVE            是否启用数据库
  74.  
  75. 自定义配置常量
  76. SYS_MODE                 开发模式， development testing pruduction
  77. IS_PATH_URL             是否使用 path url
  78. IS_SECURITY             是否开启安全过滤，防止 xss和csf攻击
